# Salesforce + Python Integration 🔗

This project connects Python with Salesforce using the Simple-Salesforce API.

## Features
- Connects to Salesforce
- Fetches Account data
- Displays results using Python

## Tech Stack
- Python
- Salesforce API
- Simple-Salesforce

## How to Run

1. Install dependency:
   pip install simple-salesforce

2. Add your Salesforce credentials

3. Run:
   python salesforce_connect.py
